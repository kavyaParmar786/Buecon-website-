## Getting .glb files from your Drive folder:
1. Download the .glb files from the shared Drive
2. Drop them here: public/models/
3. That's it — the viewer picks them up automatically

## File size tip:
Keep each .glb under 5MB for fast loading.
Use https://gltf.report/ to compress if needed.
